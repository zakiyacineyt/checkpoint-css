# Dossier Images pour le Portefeuille

## Images requises

Placez les images suivantes dans ce dossier :

### Images de Projets (1200x900px recommandé)
- `projet-ecommerce.jpg` - Capture d'écran du site e-commerce
- `projet-productivite.jpg` - Capture d'écran de l'application de productivité
- `projet-agence.jpg` - Capture d'écran du site agence créative
- `projet-saas.jpg` - Capture d'écran de la plateforme SaaS
- `projet-blog.jpg` - Capture d'écran du blog technologique
- `projet-mobile.jpg` - Capture d'écran de l'application mobile

## Notes

- Les images doivent être en format JPG ou PNG
- Taille recommandée pour les images de projets : 1200x900px (aspect ratio 4:3)
- Taille recommandée pour la photo de profil : 400x500px

## Sources d'images gratuites

Si vous n'avez pas d'images, vous pouvez en télécharger sur :
- https://unsplash.com
- https://pexels.com
- https://pixabay.com

Cherchez des termes comme "website", "app", "technology", "developer", etc.
