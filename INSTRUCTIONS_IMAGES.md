# INSTRUCTIONS POUR AJOUTER VOS IMAGES

## Photo de Profil

Placez un fichier nommé **`profile.jpg`** à la racine du dossier `checkpoint-css/`
- Dimensions recommandées : 400x500px
- Format : JPG ou PNG
- Le fichier doit être nommé exactement "profile.jpg"

## Images de Projets

Placez les 6 images de projets dans le dossier **`images/`** :

1. `images/projet-ecommerce.jpg` - Site E-commerce
2. `images/projet-productivite.jpg` - Application de Productivité
3. `images/projet-agence.jpg` - Agence Créative
4. `images/projet-saas.jpg` - Plateforme SaaS
5. `images/projet-blog.jpg` - Blog Technologique
6. `images/projet-mobile.jpg` - Application Mobile Web

Dimensions recommandées : 1200x900px (aspect ratio 4:3)
Format : JPG ou PNG
